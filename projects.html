Javascript Stunt Car

Project Description
Transformation of a 3D Viewing Pipeline to 3D dimensional points, in such a way as to compute their corresponding canvas/viewport coordinates, that would allow us to “draw” a three-dimensional line/shape/polygon, by essentially drawing the same type of geometric shape but using the projected points onto 2D canvas (i.e. “viewport”) coordinates.
Write a program that visualizes a 3D scene, as viewed from a movable camera, and use 2D Canvas drawing operations to construct a 2D projected view of it on your screen.
Features
Automatic moving camera
Hierarchical model
Parametric curve
Projection transformation
